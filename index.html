Metacritic Scraper

A simple-to-use script that scrapes Metacritic for game scores. It is coded in Python, and uses no external libraries save for httplib2. The script can receive input from the console, a text file, or a CSV, and can output to the same formats.

Improvements shall be made as time goes on, however the author will most likely convert this script into a web application using either Flask, Bottle.py, or Django.

Usage

The four acceptable command-line arguments are shown below:

python main.py -out
python main.py -csv <location of csv file>
python main.py -text <location of text file>
python main.py -help

Input and Output

The script accepts the following input formats:

  1. A text file containing the names of games and consoles, one on each line, separated by a colon (game:console).

  2. A CSV containing two columns: the first listing the names of games; the second listing the games' respective consoles.

  3. Input into the console (or terminal) in the same form as text file input.
